# Hello React Front End

> This is a Front-End React app that displays the random greeting fetched from Rails API. 

# Screenshot
![screenshot](screenshot.png)

# Built With

- React
- Redux

# Getting started

### Setup
- Clone or download this repo on your machine
- Enter project directory
- Run `npm install`
### Usage
- Run `npm start`
